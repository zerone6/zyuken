# 회의 요약 (2025.11.22)

## 1. 음성 및 접속 확인

-   초반에 마이크 문제로 몇 분 조정 후 정상 연결.

## 2. 사립 평안 관련

-   곧 헤간학교 개별 면담 예정.
-   면담에서 "결석만 없으면 괜찮다"는 식의 메시지가 있는지 확인 필요.
-   평안은 확약 방식이 아니라 **일반 시험**이 필요.
-   면담 후 결과 다시 공유 예정.

## 3. 사립 일반 입시

-   지원 희망 학교: 일본대학 제2고, 주오대 스기나미.
-   최근 모의고사 성적:
    -   3과목: 편차치 61
    -   5과목: 편차치 60
-   내부 데이터 기준:
    -   주오대 스기나미: 합격권 보더 도달.(도전권이라는 의미)
    -   일본대학 제2고: 합격 가능성 매우 높음.

## 4. 도립 고교 지원 방향

-   추천 + 일반 병행 예정.
-   최종 내신 발표 후 학교 결정.
-   현재 고민 중인 학교:
    -   도립 국제
    -   도립 신주쿠
    -   도립 토야마
    -   도립 타치카와 국제(난도 높음)
-   국제고는 IB 운영으로 장점 있으나 통학거리 부담.
-   성적으로는 도전적일 수 있으나 매력적 옵션.

## 5. 성적 관련 설명

-   최근 모의고사 편차치가 기대보다 낮아 보이는 이유:
    -   2학기 이후 수업 내용(난도 높은 심화)이 모의고사 출제와 바로 연결되지 않음.
    -   실력 저하가 아니라 **학습 구조상 점수가 떨어지는 시기**.
-   외부 사이트의偏差치와 다르더라도 **학원 내부 표준 모의고사 기준이 더 정확**함.

## 6. 주요 도립학교 편차치 기준(학원 내부 데이터)

  학교            편차치 기준
  --------------- ---------------------
  토야마            62 이상
  신주쿠          60--62
  타치카와 국제   67--69 필요
  도립 국제(IB)   난도 높고 인기 높음

## 7. 기타 의견
-   학생 태도 매우 좋음.
-   최종 내신 발표 후志望교 확정 예정.
-   평안 면담 결과 빠르게 공유 요청.

## 8. 결론

2.  도립: 신주쿠·토야마가 현실적 / 국제는 통학·난도 모두 고려 필요\
3.  평안 면담 후 정보 공유
