# 사이타마 / 치바 서부 수익형 아파트 투자 기준 정리

## 전제

-   대상: 신축 1동 아파트
-   지역: 사이타마 / 치바 서부
-   비교: 역 근접(도보 7분 이내) vs 원거리(도보 12\~15분)
-   목적: 매입 스크리닝 기준선 정리

------------------------------------------------------------------------

## 시장 현실 요약

현재 시장 대략 범위

  지역        수익률
  ----------- -----------
  도심 23구   5.5\~6.0%
  외곽 23구   6.0\~6.3%
  도쿄 외곽   6.3\~6.8%

따라서

-   **23구 신축 6.5%는 거의 어려움**
-   사이타마 외곽 또는 역거리 증가로 수익률 확보

# 2. 실제 투자 검토 우선순위

4.  和光市

## 치바 서부

2.  行徳 / 南行徳

4.  浦安 / 本八幡

------------------------------------------------------------------------

# 3. 7% 수익률 가능 구간

사이타마

-   越谷
-   일부 외곽 지역
-   역 12\~15분

치바 서부

-   西船橋 15분권
-   일부 외곽

핵심권에서는 **7% 신축 거의 희귀**

------------------------------------------------------------------------

# 4. 간단 매입 컷라인

### 和光 / 戸田 / 浦安 / 本八幡

-   6.3% 이하 → 비쌈
-   6.4\~6.6% → 정상
-   6.7% 이상 → 좋은 딜 가능

### 朝霞 / 妙典 / 行徳 / 南行徳

-   6.0% 초반 → 일반적
-   6.4\~6.6% → 좋은 물건
-   6.7% 이상 → 적극 검토

### 越谷

-   역 10분 6.5%
-   역 15분 7% 가능

------------------------------------------------------------------------
