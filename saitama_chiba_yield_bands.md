# 사이타마·치바 서부 신축 1동 아파트 수익률 밴드

> 주의: 일본에서 **실거래(클로징) 수익률 분포**는 공개 데이터가 매우 제한적입니다.  
> 이 문서는 **공개 매물 시장, 공시지가, 역세권 임대료, 현재 시장 체감**을 합쳐 만든  
> `asking yield / screening band` 자료입니다.  
> 즉 **'거래 완료 수익률 통계'가 아니라, 실전 필터링에 쓰는 호가 기준 밴드**입니다.

## 밴드 해석
- **보수적(비쌈)**: 급지는 좋지만 가격이 세다
- **시장(현실적)**: 요즘 실제로 가장 많이 만나는 구간
- **우수(깊게 볼 가치 있음)**: 숫자가 좋아서 더 조사할 만한 구간
- **의심/예외**: 너무 높은 경우, 위치/형상/설계/임대료 가정 재확인 필요

## 3) 실제 필터링 규칙

### 1호 법인 / CF 200만대가 목표일 때
- `和光 / 戸田 / 浦安 / 本八幡`
  - 6.2% 이하: 대체로 비쌈
  - 6.3~6.5%: 현실적
  - 6.6% 이상: 자세히 볼 가치 있음
- `朝霞 / 妙典 / 行徳 / 南行徳`
  - 6.1~6.3%: 일반적
  - 6.4~6.6%: 좋음
  - 6.7% 이상: 적극 검토
- `越谷 / 浦和美園`
  - 6.5% 전후도 가능
  - 7%대는 충분히 가능하나, 입지 질과 향후 수요를 더 엄격히 봐야 함

### 실전 해석
- **역 10분 이내, 신축, 7%대**를 사이타마 중심권이나 치바 서부 핵심권에서 찾는 것은 대체로 어렵다.
- **7%는 ‘중심권 초근접’이 아니라 ‘조금 더 멀거나, 토지가 약간 가벼운 축’**에서 나온다.
- 따라서 1호 법인이라면  
  `완전 핵심역 5% 후반` 과 `조금 멀지만 6.5~6.8%` 사이에서 선택하는 경우가 많다.
